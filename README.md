[![Linux Build](https://github.com/fklemme/project_template/workflows/Linux/badge.svg)](https://github.com/fklemme/project_template/actions?query=workflow%3ALinux)

C++ project template

## Build with CMake

    $ cmake -S . -B build
    $ cmake --build build

## Run tests

    $ cd build && ctest
